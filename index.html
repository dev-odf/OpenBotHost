Create a server in the panel, upload your code (or connect your Git repo), set the startup command (for example node index.js), add your DISCORD_TOKEN as an environment variable, then start the server.

What languages are supported? +

Any language that can run on Linux is supported, including Node.js, Python, Java, and Go. Libraries like Discord.js, Pycord, and Nextcord work out of the box.

Can I host multiple bots? +

Yes, as long as you stay within your plan's resource limits. For best stability, run each bot in its own server/container.
